## DecodeLabs Java Project 2 - Student Grade Calculator

### Project Description

This Java application calculates a student's total marks, average percentage, and grade based on marks entered for multiple subjects.

### Features

* Accepts marks for multiple subjects
* Calculates total marks
* Calculates average percentage
* Assigns grades automatically
* Displays formatted results

### Technologies Used

* Java
* Scanner Class
* Loops
* Conditional Statements

### Grade Criteria

| Percentage   | Grade |
| ------------ | ----- |
| 90 and above | A     |
| 80 - 89      | B     |
| 70 - 79      | C     |
| 60 - 69      | D     |
| Below 60     | F     |
